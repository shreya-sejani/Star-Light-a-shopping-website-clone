# Star-Light-a-shopping-website-clone

## Overview
This project is a comprehensive clone of a modern shopping website, designed to emulate the functionality and user experience of an e-commerce platform. It includes various functionalities such as homepage, cart, search, and checkout. 


## Features
1. **Homepage**: 
   - **Product Display**: Showcases a variety of products with images, descriptions, and prices.
     
    <img src="homepage0.png" alt="Homepage0" width="600">
    <img src="homepage1.png" alt="Homepage1" width="600">
    <img src="homepage2.png" alt="Homepage2" width="600">


2. **Product Search**: 
   - **Search Bar**: Allows users to search for products by name or category in real-time.
   - **Search Results**: Dynamically updates to display relevant products as users type.

    <img src="search.png" alt="search" width="600">
 
3. **Cart**:
   - **Add to Cart**: Users can add products to their cart.
   - **Real-Time Updates**: Cart updates in real-time as items are added or removed.

    <img src="cart.png" alt="cart" width="600">

  
4. **Checkout**:
   - **Form Validation**: Robust form validation to ensure accurate user information during the checkout process.

    <img src="checkout.png" alt="checkout" width="600">
  

5. **Responsive Design**:
  - **Cross-Device Compatibility**: Optimized for various devices, ensuring a seamless user experience on desktops, tablets, and mobile phones.

## Technologies Used
- **HTML**: For structuring the content on the web pages.
- **CSS**: For styling the web pages to make them visually appealing.
- **JavaScript**: For adding interactivity to the website.
- **AngularJS**: For backend functionality and managing dynamic data.
